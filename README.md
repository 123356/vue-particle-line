## vue-particle-line

### How to use
```
npm install vue-particle-line --save
```

### Main.js file
```javascript
import Vue from 'vue'
import vueParticleLine from 'packages/index.js'
Vue.use(vueParticleLine)
```

### App.vue file - Simple example
```html
<template>
  <div id="app">
    <vue-particle-line>
      <router-view />
    </vue-particle-line>
  </div>
</template>
```
